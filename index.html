설명형 개념·원리를 독자가 이해하도록 쉽게 풀어 쓰는 글
📌 핵심 개념
주관적 의견보다 객관적 사실·정보 전달이 목적. 독자가 몰랐던 내용을 알게 만드는 글.
🏗️ 글 구조
서론: 개념 제시
본론①: 정의·원리
본론②: 특징·과정
본론③: 사례·예시
결론: 의의·정리
✏️ 작성 요령
어려운 개념을 쉬운 말로 풀기. 예시·비유 적극 활용. "~이란 ~이다" 정의문으로 시작. 단계·순서 명확히.
예시 주제
인공지능이 우리 사회에 미치는 영향 / 기후 변화의 원인과 과정 / 운동이 건강에 좋은 이유
주장형 자신의 의견을 논거와 근거로 설득하는 글
📌 핵심 개념
"내 주장이 옳다"는 것을 독자가 납득하게 만드는 글. 근거 없는 주장은 의견에 불과.
🏗️ 글 구조
서론: 주장 선언
본론①: 근거 1
본론②: 근거 2
본론③: 반론 → 재반박
결론: 주장 재확인
✏️ 작성 요령
서론 첫 문장에 주장 명확히. 근거는 통계·사례·전문가 의견. 반론 처리가 핵심 — "일부에서는 ~라 하나, 그러나~"
예시 주제
교복 자율화는 필요한가 / 스마트폰 사용 제한은 정당한가 / AI 교육은 필수인가
비판형 현상·정책의 문제점을 논리로 분석하고 대안을 제시하는 글
📌 핵심 개념
단순히 "나쁘다"가 아니라 왜 문제인지 근거를 들어야 함. 반드시 대안으로 마무리.
🏗️ 글 구조
서론: 문제 현상 제시
본론①: 문제점 1
본론②: 문제점 2
본론③: 대안 제시
결론: 변화 촉구
✏️ 작성 요령
비판에서 끝나면 불완전. 문제점마다 구체적 사례 제시. 결론 또는 본론 마지막에 대안 필수.
예시 주제
과도한 입시 경쟁의 문제 / SNS 중독 문화 비판 / 자극적 유튜브 콘텐츠 문제점
대안제시형 문제의 현실적·단계별 해결 방법을 제안하는 글
📌 핵심 개념
"이렇게 하면 해결된다"는 설득이 핵심. 실행 가능성이 가장 중요한 평가 기준.
🏗️ 글 구조
서론: 문제 규정
본론①: 원인 분석
본론②③: 해결책 1·2
본론④: 기대 효과
결론: 실천 촉구
✏️ 작성 요령
해결책을 개인·학교·사회·정부 차원으로 나눠 쓰면 체계적. 누가·언제·어떻게 구체적으로.
예시 주제
저출산 문제 해결 방안 / 학교폭력 예방 방법 / 노인 고독사 해결책
비교·분석형 두 대상을 같은 기준으로 비교해 공통점·차이점을 분석하는 글
📌 핵심 개념
비교 기준 설정이 핵심. 기준이 다르면 결론도 달라짐. 서론에서 기준 명시 필수.
🏗️ 글 구조
서론: 대상·기준 제시
본론①②: 대상 A 분석
본론③④: 대상 B 분석
결론: 종합 판단
✏️ 작성 요령
효율성·비용·접근성 등 3가지 기준 미리 정하기. 같은 기준 순서로 양쪽 비교해야 일관성 생김.
예시 주제
온라인 수업 vs 대면 수업 / 생성형 AI vs 검색엔진 / 유산소 vs 무산소 운동
서사형 경험·사건을 이야기처럼 전개하며 교훈을 드러내는 글 (초·중)
📌 핵심 개념
이야기 형식이지만 교훈·주제 의식이 명확해야 논술. 단순 일기와 다름.
🏗️ 글 구조
서론: 경험 도입
본론①: 사건 전개
본론②: 갈등·깨달음
결론: 교훈·다짐
✏️ 작성 요령
생생한 묘사 활용. 결론에서 그 경험이 자신에게 어떤 의미인지 명확히. 교훈이 글 전체와 연결돼야 함.
예시 주제
독서 경험을 통한 성장 / 실패 경험과 배운 점 / 봉사활동에서 느낀 것
② 사고구조에 따른 유형
원인-결과형 왜 일어났는가(원인)→어떻게 됐는가(결과)를 논리적으로 연결하는 글
📌 핵심 개념
"A 때문에 B가 발생했다"는 인과관계가 핵심. 원인과 결과를 혼동하지 않도록 주의.
🏗️ 글 구조
서론: 현상 제시
본론①②: 원인 분석
본론③: 결과·영향
본론④: 해결 방향
결론: 전망·제언
✏️ 작성 요령
표면적 원인 말고 근본 원인까지 파고들기. 각 문단에 인과관계 문장 명시. "~ 때문에 ~이다" 패턴 활용.
예시 주제
청소년 우울증 증가 원인 / 저출산의 원인과 사회적 영향 / 미세먼지 발생 원인과 건강 피해
문제-해결형 문제 상황을 분석한 후 현실적 해결책을 단계별로 제시하는 글
📌 핵심 개념
대안제시형과 유사하지만 문제 분석에 더 비중을 둠. 문제를 정확히 파악해야 해결책도 정확.
🏗️ 글 구조
서론: 문제 제기
본론①: 문제 분석
본론②③: 해결책
본론④: 기대 효과
결론: 실천 호소
✏️ 작성 요령
해결책의 실행 가능성 반드시 검토. 단기·장기 해결책 구분하면 더 체계적. 막연한 "노력해야 한다" 금지.
예시 주제
디지털 중독 해결책 / 학교 내 갈등 해소 방안 / 지구온난화 대응 방법
찬반형 찬성·반대 양쪽을 검토한 뒤 자신의 최종 입장을 논리적으로 선택하는 글
📌 핵심 개념
반대 의견을 일부러 약하게 쓰면 설득력 하락. 반대 의견을 공정하게 소개한 뒤 논리로 넘어야 함.
🏗️ 글 구조
서론: 쟁점 제시
본론①②: 찬성 논거
본론③: 반대 논거
본론④: 재반박
결론: 입장 확정
✏️ 작성 요령
반대 논거를 충분히 인정한 뒤 재반박. "물론 ~이라는 시각도 있다. 그러나 ~이기 때문에" 패턴 활용.
예시 주제
원격수업 확대 찬반 / 사형제 폐지 찬반 / 인공지능 면접 도입 찬반
자료해석형 그래프·통계·기사 등 자료를 분석해 논지를 전개하는 글 (고등)
📌 핵심 개념
자료를 그대로 설명하는 게 아니라 의미를 해석·분석해야 함. "수치가 높다"가 아닌 "이것이 의미하는 바는~"
🏗️ 글 구조
서론: 자료 소개
본론①: 자료 해석
본론②: 원인 분석
본론③: 시사점
결론: 제언
✏️ 작성 요령
수치의 증감 원인 분석. 다른 자료와 연결해 종합적 해석. "자료에 따르면 ~이며 이는 ~을 의미한다" 패턴.
예시 주제
인구 감소 그래프 분석 / 청소년 스마트폰 사용 통계 해석 / 에너지 소비 변화 추이 분석
통합형 여러 자료·관점을 연결해 종합적으로 사고하는 고난도 논술 (고등·대입)
📌 핵심 개념
여러 제시문의 공통 주제를 연결하는 능력이 핵심. 대입 논술의 가장 중요한 유형.
🏗️ 글 구조
서론: 통합 주제 제시
본론①②: 자료 A·B 분석
본론③: 종합·연결
결론: 통합적 판단
✏️ 작성 요령
각 자료의 핵심 논지를 먼저 파악. 자료들 사이의 공통점·차이점 발견. 단순 요약이 아닌 나의 종합 판단 제시.
예시 주제
대입 인문논술 / 사회통합형 논술 / 철학·경제·문학 제시문 통합 분석
③ 학교 현장 특화 유형
📚 독후감 논술
책의 줄거리가 아닌 주제 의식·교훈을 논술로 전개. 자신의 삶·사회와 연결하는 것이 핵심.
📰 시사 논술
시사 이슈에 대한 분석·비판·대안을 논술로 전개. 배경 지식이 논술의 질을 좌우.
🗣️ 토론 논술
찬반형 논술의 심화. 상대 논거를 직접 반박하는 형식. 토론 후 논술로 정리할 때 활용.
🌿 환경 논술
환경 문제를 원인-결과-해결 구조로 접근. 과학적 근거와 생활 실천 방안을 함께 제시.
🤝 인성 논술
가치관·도덕적 판단을 논술로 전개. 서사형+주장형 혼합 구조. 경험 사례 적극 활용.
🧭 진로 논술
자신의 미래·직업에 대한 논술. 자기성찰+대안제시 구조. 구체적인 계획과 이유 제시 필수.
🔬 과학 논술
과학적 원리·현상을 논술로 전개. 설명형+원인결과형 혼합. 전문 용어를 쉽게 풀어 쓰기.
🤖 AI·디지털 논술
AI 윤리·미디어 리터러시·디지털 시민성 주제. 비판형+대안제시형. 최신 트렌드 반영 필요.

API 키 발급 순서 (5단계)
① console.anthropic.com 접속
② Gmail로 소셜 로그인 (회원가입 불필요)
③ 왼쪽 메뉴 Billing → 신용카드 등록 → $10 충전
④ 왼쪽 메뉴 Settings → API Keys → Create Key
⑤ 생성된 키 복사 (sk-ant-... 형태, 단 한 번만 표시됨 — 반드시 복사!)
앱에 키 입력하기
앱 왼쪽 패널 상단 입력란에 복사한 키 붙여넣기 → 저장 클릭
한 번 저장하면 이 브라우저에서는 다음부터 입력 불필요
비용 안내
논술 1회 생성 약 100~200원 · 학생 글 채점 1장 약 70~150원
월 4회 수업 기준 약 8,000~20,000원 · 처음 $10(약 14,000원) 충전 권장

① 학교급 선택 (초·중·고)
② 논술 분야 선택
③ 교과 연계 선택 (선택사항)

📖 수업 지도서 — 교사 지도 방법

PDF 다운 — 각 탭 하단 빨간 버튼 클릭 → 자동으로 PDF 파일 저장 (파일명_날짜.pdf)
논술문 복사 (논술문 모범답안 탭) — 클릭하면 클립보드에 복사 → 워드/한글 문서에 붙여넣기 가능
4
모드 B — 학생 글 AI 채점 사용법
방법 ① 사진 업로드
학생이 손으로 쓴 논술지를 스마트폰으로 촬영 → JPG 파일로 PC에 전송 → 점선 박스에 드래그하거나 클릭해서 업로드 → AI 채점 시작
방법 ② 텍스트 붙여넣기
학생이 PC로 작성한 논술문 → 전체 선택(Ctrl+A) → 복사(Ctrl+C) → 텍스트 붙여넣기 탭 선택 → 붙여넣기(Ctrl+V) → AI 채점 시작
⚠️ 주의: AI 채점은 참고용입니다. 최종 평가는 교사가 직접 판단해 주세요. 루브릭 탭의 클릭식 채점표를 활용하면 교사가 직접 점수를 입력할 수 있습니다.
💡
학교에서 교사들이 API 키 하나로 함께 쓰는 방법
핵심 원리: API 키 1개로 여러 선생님이 사용할 수 있습니다. 비용은 키를 발급한 교사 계정 하나로 합산 청구됩니다. 키를 알려주면 각자 본인 브라우저에 입력해서 사용합니다.
방법 1 키를 구두·메신저로 직접 공유 — 가장 간단
담당 교사 1명이 console.anthropic.com에서 API 키를 발급받고 카드를 등록합니다.
발급된 키(sk-ant-...)를 학교 내부 메신저(아이엠스쿨, 카카오톡 단체채팅)로 교사들에게 전달합니다.
각 교사가 앱 왼쪽 API 키 입력란에 붙여넣기 → 저장 클릭. 이후 그 컴퓨터에서는 다시 입력 불필요.
주의: 키를 외부(SNS, 이메일 공개 첨부 등)에 노출하면 타인이 무단 사용할 수 있습니다. 반드시 학교 내부 채널로만 공유하세요.
방법 2 키가 미리 입력된 HTML 파일 배포 — 더 편리
index.html 파일을 메모장으로 열고 1202번째 줄 부근에서 아래 코드를 찾습니다.
const WORKER_URL = ... 또는 apiKey 관련 줄
상태 초기화 코드 중 apiKey: localStorage.getItem('essay_api_key') || '' 를
apiKey: 'sk-ant-여기에키입력' 로 바꿔 저장합니다.
이 파일을 USB 또는 학교 공유폴더로 교사들에게 배포합니다. 파일을 받은 교사는 키 입력 없이 바로 사용 가능합니다.
⛔ 절대 주의: 키가 내장된 HTML 파일을 GitHub·인터넷에 올리면 안 됩니다. USB·공유폴더·학교 내부망으로만 배포하세요.

Q
자주 묻는 질문
"Overloaded" 오류가 뜹니다
Claude AI 서버가 일시적으로 혼잡한 상태입니다. 앱이 자동으로 최대 4회 재시도합니다 (약 1~2분 소요). 계속되면 5~10분 후 다시 시도하세요.
API 키를 다시 입력해야 하나요?
한 번 저장하면 같은 브라우저에서는 다시 입력하지 않아도 됩니다. 다른 컴퓨터·브라우저에서는 다시 입력해야 합니다.
Claude Pro 구독 중인데 왜 API 키가 필요한가요?
Claude.ai(Pro 구독)와 API는 완전히 별개 서비스입니다. 이 앱은 API를 사용하므로 console.anthropic.com에서 별도 발급이 필요합니다.
학생 논술 사진이 외부에 저장되나요?
아닙니다. 채점을 위해 Claude API에 일시 전송되지만 어디에도 저장되지 않습니다.
PDF 변환이 너무 오래 걸립니다
콘텐츠가 많을수록 시간이 걸립니다 (약 10~30초). 오류가 날 경우 인쇄하기 버튼 → 대상을 "PDF로 저장" 선택 후 저장하면 됩니다.
