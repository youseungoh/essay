## ✨ 주요 기능

### 모드 A — 논술 생성
학교급 · 분야 · 유형을 선택하면 **4가지 산출물을 동시에 자동 생성**합니다.

| 탭 | 산출물 | 활용 |
|---|---|---|
| ① 논술 예시문 | 서·본·결 구조 색상 표시 | 교사 참고 / 학생 분석용 |
| ② 학습지 | A4 인쇄용, 개요표·줄글씨칸·체크리스트 | 학생 배포 |
| ③ 루브릭 | 5영역 100점, 클릭식 인터랙티브 채점 | 교사 채점 |
| ④ 학습 가이드 | 논술 유형별 서·본·결 지도법, 발문 TIP | 교사 수업 준비 |

### 모드 B — 학생 글 AI 채점
두 가지 입력 방식을 지원합니다.

| 방식 | 설명 |
|---|---|
| 📷 사진 업로드 | 학생 손글씨 답안 사진 → AI 자동 인식 후 채점 |
| 📋 텍스트 붙여넣기 | 워드·한글 문서에서 복사 후 붙여넣기 → 즉시 채점 |

5개 영역 100점 기준으로 **자동 채점 + 영역별 피드백 + 개선 포인트** 제공

## 🚀 시작하기

### 방법 1 — 로컬에서 바로 실행 (가장 간단)

1. `index.html` 다운로드
2. 브라우저에서 파일 열기 (더블클릭 또는 드래그앤드롭)
3. API 키 입력 후 바로 사용

### 방법 2 — GitHub Pages 배포 (여러 교사와 공유)

1. GitHub 저장소 생성 → `index.html`, `.gitignore`, `README.md` 업로드
2. 저장소 **Settings → Pages → Branch: main → Save**
3. 약 1분 후 URL 생성 → 교사들과 공유

## 🔑 API 키 설정

### 발급 방법

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. Gmail로 소셜 로그인 (신용카드 불필요)
3. **Billing → 카드 등록 → $10 크레딧 충전**
4. **Settings → API Keys → Create Key**
5. 생성된 키 복사 (단 한 번만 표시됨!)

> 💡 처음 충전 권장: **$10** (약 14,000원)  
> 교사 1인 월 4회 수업 기준 약 **$6~14** 소요

### 앱에서 키 입력

앱 좌측 패널 상단 API 키 입력란에 붙여넣기 → **저장** 클릭

입력한 키는 **브라우저 localStorage에만 저장**됩니다.  
`index.html` 파일에는 기록되지 않으며, GitHub에 올려도 노출되지 않습니다.

## ⚡ Overloaded 오류 자동 처리

Claude API 서버가 혼잡할 때 자동으로 재시도합니다.

| 시도 | 대기 시간 |
|---|---|
| 1회 재시도 | 15초 후 |
| 2회 재시도 | 30초 후 |
| 3회 재시도 | 45초 후 |
| 4회 재시도 | 60초 후 |

재시도 중에는 화면 하단에 알림이 표시됩니다.  
4회 이후에도 실패하면 "잠시 후 다시 시도해주세요" 메시지가 표시됩니다.

## ❓ 자주 묻는 질문

**Q. Claude Pro 구독이 있는데 API 크레딧도 별도로 사야 하나요?**  
A. 네. claude.ai(Pro)와 API Console은 완전히 별개입니다. console.anthropic.com에서 별도 크레딧을 구매해야 합니다.

**Q. 다른 교사가 같은 파일을 열면 내 API 키로 사용되나요?**  
A. 아닙니다. API 키는 각 교사의 브라우저에 개별 저장됩니다. 다른 브라우저에서는 키를 새로 입력해야 합니다.

**Q. 학생 사진이 외부 서버에 저장되나요?**  
A. 아닙니다. 사진은 채점을 위해 Claude API에만 일시 전송되며, 어디에도 저장되지 않습니다.

**Q. API 키가 노출된 것 같으면 어떻게 하나요?**  
A. 즉시 console.anthropic.com → Settings → API Keys에서 해당 키를 삭제하고 새 키를 발급받으세요.

**Q. Overloaded 오류가 계속 발생하면?**  
A. Claude API 서버가 일시적으로 혼잡한 상태입니다. 앱이 자동으로 재시도하지만, 계속되면 5~10분 후에 다시 시도하세요.
